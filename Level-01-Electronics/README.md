# Level-01 Electronics

This level introduces the fundamental electronic components used in embedded systems and robotics. Every component is studied from first principles before being used in projects.

---

# Components Covered

## 1. Arduino UNO

### Purpose

Arduino UNO is a microcontroller development board used to control electronic components through software.

### Uses

- Read sensor data
- Control LEDs
- Drive motors
- Communicate with modules
- Execute embedded programs

---

## 2. Breadboard

### Purpose

A breadboard is used to build electronic circuits without soldering.

### Internal Structure

- Power rails distribute power along the sides.
- Every group of five holes is internally connected.
- The center gap electrically separates the two halves.

### Advantages

- Reusable
- Fast prototyping
- Easy debugging

---

## 3. Jumper Wires

### Purpose

Used to create electrical connections between components.

### Types

- Male to Male
- Male to Female
- Female to Female

Current Project

- Male to Male wires were used.

---

## 4. LED (Light Emitting Diode)

### Purpose

Converts electrical energy into light.

### Identification

- Long Leg → Anode (+)
- Short Leg → Cathode (-)
- Flat Edge → Cathode

### Important Facts

- Polarized component
- Current flows only from Anode to Cathode
- Reverse connection prevents the LED from glowing

---

## 5. Resistor

### Purpose

Limits current flowing through a circuit.

### Important Facts

- Non-polarized component
- Can be connected in either direction
- Position before or after the LED does not matter in a series circuit
